# ChatBot 桌面版聊天测试程式

[![Build Status](https://travis-ci.org/chunwei/chatbot-client.svg?branch=master)](https://travis-ci.org/chunwei/chatbot-client)

## 初始化
`npm install`

## 运行
`npm start`

## 打包

#### 当前平台(我的是win32-x64)
`npm run pack`
#### mac
`npm run pack-mac`
#### windows 32 bit
`npm run pack-win-32`

生成的可执行文件在与此工程目录同级的dist目录中

## 生成windows安装包
`npm run winstall`
